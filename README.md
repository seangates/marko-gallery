# marko-gallery

This is a demonstration of [Marko](https://markojs.com)'s capabilities. This will turn into a tutorial very soon.

## Installation

```
$ yarn
```

## Start the demo

```
$ yarn start
```

## Testing (coming soon)

```
$ yarn test
```
